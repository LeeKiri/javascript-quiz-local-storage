# javascript-quiz-local-storage
This is a timed multiple choice quiz about JavaScript. This project utilizes HTML, CSS, Bootstrap, JavaScript and local Storage. 

## Decsription

This quiz starts when you click the start button. It opens a set of multiple choice questions and starts a timer. 

When you select an answer by clicking the button it shows you the next question until the quiz is finished.

You can save your initials and score to the High Scores page. 

You can click the button to clear all the high scores or take the quiz again.

* For every correct answer you get 5 points. 
* For incorrect answers you loose 10 seconds on the timer.
